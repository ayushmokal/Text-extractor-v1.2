1. extract the zip 
2. open project directory in command prompt 
3. set environmental variables of your google-credentials ( file name "ardent-course.json" )
command :- 
set GOOGLE_APPLICATION_CREDENTIALS=path\to\your\google-credentials.json
set PORT=3000

4. in command prompt type command "npm install" 
5. to start the application type command "node app.js
